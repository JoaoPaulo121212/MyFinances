💰 MyFinance - Dashboard de Finanças Pessoais
O MyFinance é uma aplicação web leve e intuitiva projetada para ajudar usuários a monitorarem suas finanças de forma simples. O projeto foca em oferecer uma visão clara de entradas e saídas, utilizando gráficos interativos para facilitar a compreensão da saúde financeira.

🚀 Funcionalidades
Gestão de Transações: Adicione receitas e despesas com descrição e valor.

Cálculo Automático: Saldo total, entradas e saídas atualizados em tempo real através de lógica JavaScript.

Visualização por Gráficos: Gráfico de rosca dinâmico utilizando a biblioteca Chart.js para representar a proporção entre ganhos e gastos.

Persistência de Dados: Uso de LocalStorage para garantir que os dados não sejam perdidos ao recarregar a página.

Interface Responsiva: Design adaptável para diferentes tamanhos de tela (Mobile e Desktop).

🛠️ Tecnologias Utilizadas
Este projeto foi construído utilizando as tecnologias fundamentais da web, focando em manipulação de DOM e lógica de programação:

HTML5: Estrutura semântica para melhor acessibilidade.

CSS3: Layout moderno utilizando CSS Grid e Flexbox, além de variáveis para fácil manutenção de cores.

JavaScript (Vanilla): Lógica de cálculo, manipulação de arrays (map, filter, reduce) e gestão de estado.

Chart.js: Biblioteca externa para renderização de gráficos.

🧠 Desafios e Aprendizados
Durante o desenvolvimento deste projeto, foquei em resolver problemas comuns no desenvolvimento Front-end:

Gestão de Dados: Implementar a lógica de soma e subtração garantindo que os tipos de dados (String vs Number) fossem tratados corretamente.

Manipulação de DOM: Atualizar a interface do usuário de forma eficiente sempre que um novo item é adicionado ou removido.

Experiência do Usuário (UX): Garantir que o gráfico seja limpo e que a interface "respire", evitando poluição visual.
